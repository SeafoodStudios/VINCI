VINCI Language

VINCI is a lightweight, numbers oriented and Turing complete programming language designed for general purpose use and beginners. It serves as a good transition from higher to lower level languages and is good for educational demonstrations of calculations. It brings us to retro, old fashioned programming, without that much of a learning curve.

Educational

VINCI is great for educational challenges and programs, because of its restraints. It only has JUMP commands for control flow, making it a great way to make students think.

Simple

VINCI has a very minimal, readable syntax, therefore having a smaller learning curve. But, you can do a lot with the syntax, because it is Turing complete, meaning it can compute anything given enough resources and time.

Creativity

VINCI is not only usable in educational situations, but also for fun! It can be used in the esoteric community, because of how strict the language is. There are infinite possibilities, waiting to be made possible.
